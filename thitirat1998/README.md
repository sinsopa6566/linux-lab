# วิธีรันโปรแกรม

## 1.ติดตั้งโปรแกรมจำเป็น
```sh
(sudo) pip install lxml
(sudo) pip install BeautifulSoup4
```
## 2.เริ่มใช้งานโปรแกรม ให้ clone project
```sh
cd thitirat1998
```
## 3. ติดตั้ง package ที่จำเป็น
```sh
pip install --user click requests Pillow
```
## 4. ติดตั้ง
```sh
pip install --user .
```
